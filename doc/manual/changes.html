Version 3.0

What's New?

  • Adapted to also work in Lua 5.2.
  • New module <%=link("Modules.proto")%> to support <%=link("Basics.prototyping", "prototyping")%>.
  • Classes adopt a basic model that allow the program to manipulate any LOOP class in a uniform way, see <%=link("Classes.classops")%>.
  • New module with implementations of the <%=link("Basics.__new", "__new")%> metamethod.
  • New utilitary function <%=link("Modules.table.memoize")%> to create memoize tables.

What's Changed?

  • Preloader is distributed as a separate <%=link("LuaPreloader", "package")%>.
  • LOOP Class Library is distributed as a separate <%=link("LOOP.ClassLib", "package")%>.
  • LOOP modules for component-based development are distributed as a separate <%=link("LOOP.Component", "package")%>.
  • The following module functions were renamed for clarity:
    LOOP 2.x   LOOP 3.0
    classof   <%=link("Modules.base.getclass")%>
    instanceof   <%=link("Modules.base.isinstanceof")%>
    memberof   <%=link("Modules.base.getmember")%>
    subclassof   <%=link("Modules.simple.issubclassof")%>
    superclass   <%=link("Modules.simple.getsuper")%>
